Pokemon Egg Generator

About

Welcome to the Pokemon Egg Generator. It randomly generates level 1 Pokemon with moves, abilities, natures, etc. in the Pokemon Showdown format, which can then be imported into PKHeX. Using the options below, you can control the number of Pokemon to generate, what Generation they are compatible with, and whether or not you want them to be shiny, have hidden abilities, or egg moves.

This tool is still in development. If you find a bug, please log the issue here.
